# React Real Estate UI Design

A simple **Real Estate website design** built using **ReactJS, HTML, CSS, and SCSS**.

## 🚀 Features

### Homepage [/]

- Navigation menu
- Search functionality

### Listpage [/list]

- Location wise results in list format
- Filter funcationality
- Map with listings

### Singlepage [/1]

- Photos of house, description, price
- Location map
- Sizes and nearby places

### Profilepage [/profile]

- User information
- Saved lists
- Messages and chat window

## 🛠️ Technologies Used

Frontend:

- REACTJS
- HTML
- CSS
- SCSS

## 📦 Installation

1. Clone the repository

<!-- ```
git clone https://github.com/yourusername/crm-system.git
``` -->

2. Go into project folder
<!-- ```
cd project

````-->

3. Install dependencies
<!-- ```
npm install
``` -->

4. Configure environment variables
<!-- ```
copy .env.example .env
``` -->

5. Start development server
<!-- ```
npm run dev
``` -->

6. Run the project
<!-- ```
http://localhost:5173
``` -->
